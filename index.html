Optimal Faculty-Course Assignment Using Graph Theory

Developed an optimization solution to efficiently assign faculty to course sections. This project used graph theory by constructing a weighted bipartite graph where edges represented historical teaching data and course relevance. By applying the Linear Sum Assignment algorithm, the system generated optimal pairings to improve the assignment process.

Residential City Recommender System

Designed and developed a data-driven system to recommend optimal residential cities based on a user's workplace or school. This project collected real-time data from the Google Maps API and modeled cities as a weighted graph. It then utilized multiple shortest path algorithms, including Dijkstra’s, brute-force search, and dynamic programming, to compute and compare optimal commute routes. Additionally, clustering and divide-and-conquer strategies were implemented to ensure the system's scalability and efficiency.
